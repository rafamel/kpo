Optional cwd

cwd: undefined | string

Inherited from IScopeOptions.cwd

  • Defined in types.ts:104

Used for relative paths resolution and as default cwd on command execution. It is also the directory scopes are resolved from. By default, it will be the directory in which a sibling or parent package.json is found for the kpo scripts file, or the directory of the kpo file itself otherwise, unless a explicit directory is passed on the cli -see ICliOptions. If it is a relative path itself, it will be resolved:

  • relative to the process cwd if passed on the cli.
  • relative to the package.json if set as an option there.
  • relative to the kpo scripts file if set there.
